06 / Motion

Movimento com freio

Smooth, sem exagero. Zero bounce, spring ou overshoot. Transições padrão 300ms ease-out. prefers-reduced-motion respeitado via gate JS.

Reveal on scroll — fade + translateY 24px, 800ms ease-out, trigger via IntersectionObserver a 15% de visibilidade.
Glow pulse — aplicado só em CTAs críticos. Box-shadow oscilando entre 60% e 90% de opacidade, ciclo 2.5s.
Shine sweep — gradient branco 20% atravessa o botão em 600ms no hover.
Card border glow — opacity 0→1 do gradient border no hover, 400ms.
Hover translate — cards sobem 4px, botões 2px, transição 300ms.
Rotate slow — dashed ring da guarantee badge gira 360° em 40s loop infinito.
Countdown JS — atualiza a cada 1s quando entra no viewport, com fallback se prefers-reduced-motion.
Gate motion — classe html.js-motion só é adicionada se usuário não pediu reduced motion. CSS gateia tudo por ela.
